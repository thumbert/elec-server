Customer counts for NGrid

  • Get the historical usage of the town of Attleboro, for all customer classes (utility and competitive supply)
    customercounts/v1/kwh/town/Attleboro
  • Get the historical usage for each of the towns in zone SEMA, for rate class R1
    customercounts/v1/kwh/zone/SEMA/rateclass/R1
  • Get all the towns in the collection
    customercounts/v1/towns
  • Get all the zones in the collection
    customercounts/v1/zones
  • Get all the available rate classes
    customercounts/v1/rateclasses

DAM binding constraints

  • Get all the occurrences of a given constraint
    bc/v1/market/da/constraintname/BRIGHTON329-530 A LN
  • Get all the constraints between a two dates
    bc/v1/market/da/start/20170101/end/20170102

EPA emission data

  • Get historical emission data by facility. Each power plant is specified by a facility id and an unit id. The query returns data for all units in the facility when there was CO2 emitted. For older periods of time, there may be NOX data only for a given day. This query will not return that data.
    epa_emissions/v1/fac_id/328/start/20170101/end/20170131
  • Get the EPA emission data by power plant. Each power plant is specified by a facility id and an unit id. To get the existing ids, see the [/units] route below.
    epa_emissions/v1/fac_id/328/unit_id/1083/start/20170101/end/20170131
  • Get the unique combination of facility id and unit id in the database.
    epa_emissions/v1/units
  • Get all the facilities tracked in the database. Note that the [FACILITY_NAME] is not unique.
    epa_emissions/v1/facilities
  • Get the daily CO2 emissions for facilities that are subject to the MA GHG 310 CMR 7.74 ruling. The [CO2_MASS] value is in short tons.
    epa_emissions/v1/ma_compliance/start/20170101/end/20171231

Masked DA demand bids

  • Get the total MWh demand bids by day and load zone for given a participant between two dates.
    da_demand_bids/v1/mwh/participantId/206845/start/20170701/end/20171001
  • Get the total MWh demand bids by day, participant, and load zone between two dates.
    da_demand_bids/v1/mwh/participantId/206845/ptid/4004/start/20170701/end/20171001
  • Get the total MWh demand bids by day and participant between two dates (aggregate over the load zones).
    da_demand_bids/v1/mwh/participantId/206845/start/20170701/end/20171001

Masked DA energy offers

  • Get stack for a given date and hour ending. For the extra hour in the fall, enter '02X' same as the ISONE. Only the offers of the units that are available are returned, sorted by price.
    da_energy_offers/v1/date/stack/20170701/hourending/16
  • Get all the generation offers for a given date and hour ending. For the extra hour in the fall, enter '02X' same as the ISONE.
    da_energy_offers/v1/date/20170701/hourending/16
  • Get all the daily data for all the generators for a day
    da_energy_offers/v1/daily_data/day/20170701
  • Get one variable for one generator between a start and end date, in this example the 'Economic Maximum'
    da_energy_offers/v1/assetId/41406/variable/Economic Maximum/start/20170701/end/20171001
  • Get one variable for all the generators between a start and end date, in this example the 'Economic Maximum'
    da_energy_offers/v1/variable/Economic Maximum/start/20170701/end/20171001

Ptid table

  • Get the latest ptid table (node name, zone, etc.)
    ptids/v1/current
  • Get the ptid table available as of a given date
    ptids/v1/asofdate/20170810
  • Get the dates when the ptid table was modified
    ptids/v1/dates
